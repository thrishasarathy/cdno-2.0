Project Type:
Static web application with localStorage-based authentication and personalized notification dashboard.


Folder Structure:
- index.html
- login.html
- signup.html
- profile.html
- notifications.html
- css/style.css
- css/dashboard.css
- js/script.js
- js/auth.js
- js/database.js
- js/dashboard.js
- assets/logo.jpg


How to Run:
1. Keep all files in the correct folder structure.
2. Add your logo image as:
   assets/logo.jpg
3. Open index.html in browser.
4. Login using:
   user1@gmail.com / 1234
   user2@gmail.com / 1234
   user3@gmail.com / 1234


Features:
- Landing page
- Local login / signup
- Protected dashboard
- User profile
- Notification ranking engine
- Context mode switching
- Search / filter / sorting
- Keyword boost
- Pin / delete actions
- Optimization history
- Mobile responsive design


Deployment:
Can be directly deployed on Netlify as a static website.


Important:
Before first demo, open browser console and run:
localStorage.clear();
sessionStorage.clear();


Then refresh and login again.
